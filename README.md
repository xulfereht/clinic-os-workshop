# Clinic-OS Workshop Environment

AI 에이전트 기반 한의원 웹사이트 구축 워크숍 환경입니다.

## 시작하기

1. **Code** 버튼 → **Codespaces** → **Create codespace on main**
2. 환경이 준비되면 (2~3분) 터미널에서 Claude Code 실행:
   ```
   claude
   ```
3. Claude Code에게 스타터킷 설치를 요청하세요:
   ```
   HQ에서 스타터킷을 받아서 설치해줘.
   HQ URL: https://clinic-os-hq.pages.dev
   Client ID: (안내받은 ID)
   ```

## 포함된 도구

- Node.js 22
- Cloudflare Wrangler CLI
- Astro
- Claude Code (AI 에이전트)

워크숍 단톡방에 질문해주세요.
